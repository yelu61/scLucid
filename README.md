# scRNA-toolkit

A comprehensive toolkit for single-cell RNA-seq data analysis.

## Installation

```bash
# From PyPI (once published)
pip install scRNA-toolkit

# From GitHub
pip install git+https://github.com/yourusername/scRNA-toolkit.git

# For development (editable mode)
git clone https://github.com/yourusername/scRNA-toolkit.git
cd scRNA-toolkit
pip install -e .
