# scLucid

A comprehensive toolkit for single-cell RNA-seq data analysis.

## Installation

```bash
# From PyPI (once published)
pip install scLucid

# From GitHub
pip install git+https://github.com/yourusername/scLucid.git

# For development (editable mode)
git clone https://github.com/yourusername/scLucid.git
cd scLucid
pip install -e .
